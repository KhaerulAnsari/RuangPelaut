Kebijakan Privasi

Berlaku 09 Juni 2020

  1. Data digunakan untuk penyaringan dan pemetaan anggota pelaut baik yang sedang belajar ataupun yang sudah bekerja
  2. Penggunaan foto hanya untuk ditampilkan, tidak disebarkan
  3. Permintaan pengolaan storage untuk keperluan upload foto saja tidak ada download
  4. Akses foto juga hanya digunakan untuk mengirim foto ke server
  5. Penggunaan internet digunakan sebagai pengambilan data dari server untuk disajikan
  6. Ruangpelaut hanya menghubungkan antara pelaut dengan institusi pelaut agar lebih mudah, tidak ada fitur komunikasi antara keduanya
  7. Setiap data yang dikirim ke Ruangpelaut berupa CV akan diperlihatkan kepada institusi pelaut demi kelancaran pencarian kerja pelaut
